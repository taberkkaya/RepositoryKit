**Lightweight, generic repository implementation for Entity Framework Core**  
Designed for personal and professional projects with clarity, extensibility, and clean architecture in mind.

## 🚀 Features

- Provides asynchronous CRUD operations optimized for Entity Framework Core
- Supports both tracking and no-tracking query scenarios
- Minimal external dependencies to keep the package lightweight and fast
- Clean and extensible architecture that allows easy customization
- Suitable for personal projects and scalable for larger applications
- Open source and MIT licensed for maximum flexibility

## 📦 Installation

Install RepositoryKit via NuGet Package Manager Console:

```bash
Install-Package RepositoryKit
```

Or using the .NET CLI:

```
dotnet add package RepositoryKit
```

## 📝 Notes

_RepositoryKit is initially released as a functional equivalent and replacement for Taner Saydam's [TS.GenericRepository package](https://github.com/TanerSaydam/TS.GenericRepository.NuGet), tailored to meet the needs of my personal projects. Future versions will expand on this foundation with additional features and customization options._
